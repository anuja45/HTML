MINI PROJECT-I

Name:AI TIC TAC TOE APP

“Tic Tac Toe Game” is fun game in which two players alternatively places X’s and O’s on 3x3 grid. This game is played on a wooden board, a table or piece of paper. Now this is world of games on mobile or PC. Hence this project is made to make an Android App to play the game.Tic Tac Toe game is played on 3x3 grid. The player who succeeds in placing three of their marks in a horizontal, vertical or diagonal row wins the game. The two players are user and AI. In this project first user will enter X’s first then it is turn of AI. Player who will form horizontal, vertical or diagonal of three marks wins the game else the game will be draw.

App is developed in java language and Android Studio IDE is used. I have used Minmax algorithm and alpha beta pruning.
